# Problem Statement

Small businesses face a content creation bottleneck that directly impacts their growth:

1. **Volume demands:** Maintaining a presence across Instagram, TikTok, LinkedIn, and X requires 4â€“7 posts per platform per week â€” that's 20â€“30+ pieces of content weekly. Most small business owners simply can't keep up.
2. **Fragmented tooling:** Existing tools force owners to context-switch between 3â€“5 separate products: Jasper for copy, Canva for images, Hootsuite for scheduling, ChatGPT for ideas. Each tool produces isolated outputs that must be manually stitched together.
3. **Disconnected AI outputs:** No tool generates copy and visuals together. A caption written by one AI and an image generated by another feel generic and mismatched. There's no creative coherence â€” the text says "warm and inviting" but the image is cold and sterile.
4. **Cost barrier:** Enterprise tools (Hootsuite at $99+/mo, Jasper at $49+/mo) are priced for marketing teams, not solo operators. Small businesses need agency-quality output at DIY prices.

# Solution

Amplifi is an AI creative director that analyzes your brand and produces complete social media content packages â€” captions, images, hashtags, and posting schedules â€” all generated together in a single interleaved output stream. The copy and visuals are born together, ensuring every post feels cohesive and on-brand.

## Core Experience

- **Brand-first intelligence:** Upload your website URL or product photos. Amplifi's Brand Analyst agent builds a complete brand profile â€” colors, tone, target audience, competitive positioning â€” that informs every piece of content it creates. No more generic AI outputs.
- **Interleaved content generation:** Watch as captions and matching images stream together in real-time. Monday's post appears with its custom product photo, then Tuesday's story graphic â€” all flowing as one cohesive output. This is Gemini's interleaved output in action.
- **Full content packages:** Not individual assets but complete weekly content plans. Each post includes caption + product/lifestyle image + 5â€“8 relevant hashtags + optimal posting time + platform-specific formatting notes.
- **Multi-agent quality pipeline:** Four specialized agents work in sequence â€” Brand Analyst, Strategist, Content Creator, Reviewer â€” ensuring every post passes brand consistency checks before delivery.

## What Makes This Different

The Creative Storyteller category specifically asks for projects that "leverage Gemini's interleaved text and image output." Amplifi doesn't just use interleaved output as a feature â€” it IS the product. The entire value proposition depends on text and images being generated together:

- Captions reference visual elements that are simultaneously being created
- Images reflect the mood, tone, and messaging of the accompanying copy
- Brand colors, typography style, and aesthetic carry across both text and visuals
- A week's worth of content maintains visual and tonal consistency because it's generated in one contextual session
- For video-first platforms (Reels, TikTok, Stories), the interleaved hero image becomes the first frame of a Veo-generated video clip â€” ensuring the static post and the video clip share the same visual DNA

### Agent Orchestration

The four agents run as a **SequentialAgent** pipeline in ADK:

```
Brand Analyst â†’ Strategy Agent â†’ Content Creator (Ã—7 days) â†’ Review Agent
```

The Content Creator is called once per calendar day, producing one post per invocation. The Review Agent evaluates each post individually against the brand profile.

## Interleaved Output â€” The Star Feature

The Content Creator Agent uses Gemini's `responseModalities: ["TEXT", "IMAGE"]` to generate captions and matching product images in a single API call. This is the core differentiator.

**How it works:**
- The model receives the brand profile (colors, tone, audience) and a daily content brief ("Tuesday: showcase spring pastries, warm and inviting tone")
- It generates caption text, then inline generates a matching product/lifestyle image, then adds hashtags and posting notes â€” all in one response
- The response contains interleaved `text` and `inline_data` (image bytes) parts that stream to the frontend progressively

**Supported models:**
- `gemini-2.5-flash` with `responseModalities: ["TEXT", "IMAGE"]` (primary)
- `gemini-3-pro-image-preview` for highest quality image gen (fallback)
- Note: `gemini-2.5-flash-image-preview` has been shut down â€” do not use

**Pricing:** ~$0.039 per generated image (1,290 tokens at $30/1M output tokens). With the $100 Google Cloud credit, that's ~2,500 generated images â€” more than sufficient for development + demo + buffer.
